AI-Enabled IoT Health Monitoring System

This project is an **IoT-based smart health monitoring system** that continuously tracks vital body parameters such as **Heart Rate (BPM), Body Temperature, and SpO2 (Oxygen Level)**.  
The collected data is processed using **Machine Learning / AI** to analyze health condition and identify abnormalities in real-time.

Technology Stack

Hardware
- Arduino / NodeMCU / ESP8266 / ESP32  
- Temperature Sensor (e.g., LM35 / DS18B20)  
- Pulse / Heartbeat Sensor  
- SpO2 Sensor (MAX30100 / MAX30102)  
- Power Supply  

Software & Tools
- Arduino IDE  
- Python (for AI model)  
- Flask / Node.js (optional backend)  
- HTML / CSS / JS (Frontend)  
- Git & GitHub  



AI / Machine Learning

The system uses **Machine Learning to analyze patterns in health data** such as:
- Abnormal Heart Rate  
- Fever Detection  
- Oxygen Level Drop Prediction  

Predictions can be used for **early-stage health alerting**.
